# Content and Tone Guide

## Voice

- Confident
- Calm
- Precise
- Premium but not theatrical

## Copy Rules

- Prefer short labels over full sentences
- Lead with action in CTA copy
- Surface units and context for every metric
- Avoid ambiguous finance and health terms

## CTA Style

- Primary: verb + object, example `Proceed to Pay`
- Secondary: noun phrases, example `Order History`
- Tertiary: concise utility actions

## Errors

- Name the issue clearly
- Explain next step
- Preserve user context and entries whenever possible

## Success

- Confirm what completed
- Show key confirmation detail
- Offer one clear next action and one secondary action
